Wymagania od patryka:
Struktura HTML - Utwórz prosty szablon HTML z następującymi sekcjami:
- Nagłówek (Header): Może zawierać nazwę i logo.
- Sekcja "O mnie": Krótki opis osoby lub fikcyjnej firmy.
- Sekcja "Umiejętności": Lista umiejętności lub usług.
- Stopka (Footer): Informacje kontaktowe lub linki do mediów społecznościowych.
Stylizacja CSS - Użyj CSS do stylizacji strony. Skup się na:
- Użyciu różnych rodzajów czcionek.
- Zastosowaniu kolorów odpowiadających schematowi kolorów (można użyć narzędzi takich jak Coolors lub Adobe Color).
- Dodaniu prostych animacji lub efektów przejścia dla linków lub przycisków.
Layout z użyciem Flexboxa:
- Użyj flexboxa do organizacji layoutu sekcji i ich zawartości.
- Upewnij się, że strona jest responsywna i dobrze wygląda na różnych rozmiarach ekranu.
Dodatkowe wyzwania:
- Dodaj galerię zdjęć w sekcji "O mnie", używając flexboxa do ustalenia układu.
- Utwórz responsywne menu nawigacyjne, które będzie się dostosowywać do rozmiaru ekranu (np. hamburger menu na małych ekranach).
